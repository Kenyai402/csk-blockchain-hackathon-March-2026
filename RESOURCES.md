# Hackathon Learning Resources 📚

These resources will help you understand blockchain technology and explore tools used in Web3 development.

# 1. Blockchain Basics

### What is Blockchain?

A blockchain is a decentralized digital ledger that records transactions across multiple computers in a secure and transparent way.

Key characteristics include:

• decentralization  
• transparency  
• immutability  
• trustless systems  

---

# 2. Smart Contracts

Smart contracts are programs that automatically execute agreements when certain conditions are met.

They are commonly written using the Solidity programming language.

Learn more here:

Smart Contracts Overview  
https://ethereum.org/en/developers/docs/smart-contracts/
